# AI Data Report Generator 🚀

An AI-powered automated dataset analytics platform that transforms raw CSV files into professional data quality reports in just a few clicks.

## 🌐 Live Demo

https://ai-data-report-generator.onrender.com

---

# 📌 Problem It Solves

In real-world data science and analytics workflows, generating a complete EDA (Exploratory Data Analysis) report manually can take 1–2 hours or even more.

Analysts usually need to:

* check missing values
* detect duplicates
* analyze outliers
* calculate statistics
* evaluate data quality
* create summaries
* generate reports manually

This project automates the entire workflow.

Now users can:

1. Upload a dataset
2. Click a few buttons
3. Get a complete professional analytics report instantly

⚡ What normally takes hours now takes seconds.

# 📑 Generated Report Includes

The system automatically generates:

* Dataset Overview
* Total Rows & Columns
* Missing Value Analysis
* Duplicate Analysis
* Memory Usage
* Column-wise Analysis
* Numeric Statistics
* Mean / Median / Standard Deviation
* Outlier Detection
* Data Quality Health Score 

Example generated report:

* 55,000 rows analyzed
* 48,661 missing values detected
* 4,718 duplicate rows identified
* 1,500 outliers detected automatically
